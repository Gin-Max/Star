PREREQUIS : **NodeJS nécessaire pour lancer le programme**

aller dans le répertoire du fichier dans l'invite de commandes
exemple : "C:\Users\Maxence\Desktop\Test>"

Pour  lancer le Script, utiliser NodeJS :
exemple : "C:\Users\Maxence\Desktop\Test>node test.js"

donnez un argument au script d'une valeur > 0 (ou bien le script n'affichera rien)
exemple : "C:\Users\Maxence\Desktop\Test>node test.js 5"
